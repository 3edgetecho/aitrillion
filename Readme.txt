== Description ==
A Woocommerce plugin, integrate with AiTrillion platform

== Installation ==
1. upload aitrillion.zip into plugins
2. activate aitrillion plugin
3. go to AiTrillion -> Settings and update API key and Password (you will find this under your Aitrillion account dashboard)
